# Step 0 - Product Context Calibration
*Foundation Before Discovery*

## Your Mission
Before diving into discovery, establish crystal-clear product context. This calibrates the AI for maximum relevance across all 5 steps.

## Option 1: Structured Product Description
Use this template:

**Product Name**: [Your product name]

**Target Audience**: [Specific user groups - be precise, not "everyone"]

**Problem It Solves**: [The core pain point - one clear sentence]

**How It Works**: [Basic mechanism/approach - 2-3 sentences max]

**What Makes It Unique**: [Your differentiation - what's genuinely different]

**Current Status**: [Idea/MVP/Live product - sets discovery focus]

## Option 2: Upload Existing Documentation
"I have existing product documentation. Analyze this brief and extract: target audience, core problem, solution approach, and unique value proposition. Then summarize in the structured format above."

## Optional Quality Check Prompts
- "Is my target audience specific enough for meaningful personas?"
- "Does my problem statement reflect real user pain or assumed pain?"
- "What assumptions am I making that need validation?"

**Deliverable**: Clear product foundation + assumption list
**Timer**: 5 minutes - GO!

## Why This Matters
- Sharpens persona creation (Step 1)
- Focuses pain point analysis (Step 2)
- Guides solution ideation (Step 3)
- Targets competitive research (Step 4)
- Frames stakeholder communication (Step 5)
